# ShareMirror

「いま見えている通り」をそのまま映す **ミラーウィンドウ** を作成する、Windows向けの小さなユーティリティです。

アクティブウィンドウの表示内容を定期的に取得し、  
専用のウィンドウにリアルタイムで表示します。

他のユーザーに画面共有する際に、

- 共有するウィンドウを毎回選ぶのが面倒
- ウィンドウ共有だと右クリックメニューなどが共有されない
- モニタ全体共有だと、UWQHD 等では表示が小さくなってしまう

といった問題を回避することを目的としています。

## 開発目的

- **アクティブウィンドウを自動追従**するミラーウィンドウを作る
- 「ウィンドウ共有」ではなく **「見えている画面そのもの」を共有**しやすくする
- モニタ全体共有よりも、**見せたい内容を大きく・分かりやすく表示**する

## 特徴

- アクティブウィンドウを定期的にキャプチャし、ミラーウィンドウに表示します
- マウスカーソルや右クリックメニューなども表示します
- 安全にミラーウィンドウに表示します
    - デスクトップ画面は無効としています
    - アクティブウィンドウを切り替えて3秒間継続で切り替えします

## 動作環境

- Windows 10 / 11

## ファイル構成

- `bin` 実行ファイル `ShareMirror.exe` が入っています
- `src` ソースコード（閲覧・ビルド用。実行には不要です）

## 使用方法

1. プログラム(`ShareMirror.exe`)を起動します
2. ミラーウィンドウが表示されます
3. 現在アクティブなウィンドウの内容が、ミラーウィンドウに表示されます
4. 共有ソフトでは **このミラーウィンドウを共有**してください  
   （モニタ全体共有ではなく、ウィンドウ共有がおすすめです）

### 操作方法

- **左クリック＋ドラッグ**  
  ミラーウィンドウを移動します
- **アクティブウィンドウを切り替える**  
  3秒間継続で自動でミラー対象が切り替わります

### 終了方法

- 次のいずれかの方法で終了できます
    - ミラーウィンドウを右クリック → 「`終了`」を選択
    - ミラーウィンドウを選択 → 「`DEL`」「`ESC`」「`BACK`」のいずれかを入力

## 注意点・制限

- ウィンドウ外に大きくはみ出したメニューなどは、欠ける場合があります
- アプリや環境によっては、取得できない描画要素が存在する場合があります
- 本ツールは **画面共有の利便性向上** を目的としており、完全な画面キャプチャ再現を保証するものではありません
